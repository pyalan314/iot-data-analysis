# Output Directory

This directory contains all prediction outputs generated by the trend prediction scripts.

## Directory Structure

```
output/
├── CSV Files (Predictions)
│   ├── 1_predictions_sarimax_weekly.csv
│   ├── 2_predictions_sarimax_intraday.csv
│   ├── 3_predictions_prophet_weekly.csv
│   ├── 4_predictions_prophet_intraday.csv
│   ├── 5_predictions_lightgbm_weekly.csv
│   └── 6_predictions_lightgbm_intraday.csv
│
└── PNG Files (Visualizations)
    ├── 1_sarimax_weekly_results.png
    ├── 2_sarimax_intraday_results.png
    ├── 3_prophet_weekly_results.png
    ├── 4_prophet_intraday_results.png
    ├── 5_lightgbm_weekly_results.png
    └── 6_lightgbm_intraday_results.png
```

## Output Files

### SARIMAX Models
- **1_predictions_sarimax_weekly.csv** - Weekly predictions (168 hourly intervals)
- **1_sarimax_weekly_results.png** - Visualization with 4 subplots
- **2_predictions_sarimax_intraday.csv** - Intraday predictions (96 15-minute intervals)
- **2_sarimax_intraday_results.png** - Visualization with 4 subplots

### Prophet Models
- **3_predictions_prophet_weekly.csv** - Weekly predictions (336 30-minute intervals)
- **3_prophet_weekly_results.png** - Visualization with 8 subplots
- **4_predictions_prophet_intraday.csv** - Intraday predictions (48 15-minute intervals)
- **4_prophet_intraday_results.png** - Visualization with 6 subplots

### LightGBM Models
- **5_predictions_lightgbm_weekly.csv** - Weekly predictions (336 30-minute intervals)
- **5_lightgbm_weekly_results.png** - Visualization with 8 subplots
- **6_predictions_lightgbm_intraday.csv** - Intraday predictions (per-minute data)
- **6_lightgbm_intraday_results.png** - Visualization with 8 subplots

## CSV Format

All prediction CSV files contain:
- `timestamp` - Prediction timestamp
- `predicted_power_usage` - Predicted power usage in kW
- `lower_bound` - Lower confidence bound (Prophet models only)
- `upper_bound` - Upper confidence bound (Prophet models only)

## Notes

- Output files are automatically generated when running the corresponding scripts
- Files are gitignored to avoid tracking generated outputs
- All timestamps are in the format: `YYYY-MM-DD HH:MM:SS`
- Visualizations are saved at 300 DPI for high quality
